# 📊 Samsung_mobile_insights

Comprehensive Analysis of Samsung Smartphones – Features, Specs &amp; Trends

This project presents an interactive Tableau dashboard that provides a **comprehensive analysis of Samsung smartphones**, focusing on specifications, market distribution, and feature comparisons.

## 📌 Objective

To analyze and visualize various aspects of Samsung smartphones—such as processor types, storage options, battery capacity, RAM, OS versions, and pricing trends—to assist consumers, analysts, and business stakeholders in making data-driven decisions.

## 📈 Dashboard Features

### 🔍 Device Overview
- Dropdown menu to select specific models (e.g., OPPO Reno10 5G).
- Displays technical specifications including:
  - **Display**: Size, Type, Resolution, Refresh Rate.
  - **Network**: Supported generations (2G to 5G), Connectivity options.
  - **Processor**: OS type, Clock speed, Core count.
  - **Camera**: MP count, Lens details, Features like Portrait and Night mode.

---

### 📊 Number of Mobiles by Processor
- Horizontal bar chart representing **processor distribution** (e.g., Dimensity, Exynos, Snapdragon).
- Helps in identifying the most frequently used processors in the product line.

---

### 💾 Top 20 Mobile Storage
- Vertical bar chart displaying **internal storage capacities** among top models.
- Useful for spotting high-end and budget-friendly device categories.

---

### 🔋 RAM & Battery Trends
- Line charts for **RAM and Battery** usage trends across models.
- Shows correlation between memory and power capacity.

---

### 💰 Price Distribution
- Area chart representing **price range** of Samsung models.
- A quick glance at premium, mid-range, and budget models.

---

### 📱 Android Version Distribution
- Pie chart visualizing the **Android OS version spread** among the devices.
- Useful for understanding how many devices run older or newer Android versions.
